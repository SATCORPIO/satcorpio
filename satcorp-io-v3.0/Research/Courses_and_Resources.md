# COURSES_AND_RESOURCES // EXPERIMENTAL_LEVEL
## Master the Craft

> [!TIP]
> To truly unlock the "Midnight Neural" ecosystem, I’ve curated these top-tier resources. This isn't beginner code; this is high-fidelity creative engineering.

### 1. Three.js Journey (Bruno Simon)
- **Status:** The absolute industry standard.
- **Why:** To build the "3D Command Hub." Bruno covers everything from standard meshes to **advanced GLSL shaders**. 
- **Focus:** [Shaders & Post-Processing](https://threejs-journey.com/) (Module 5 & 6) is the key for SATCORP.

### 2. Frontend Masters - Advanced WebGL & Shaders (Matt DesLauriers)
- **Status:** Deep-dive into graphics mathematics.
- **Why:** If you want custom data-visualizations that look like "classified telemetry," you need this. It covers vector math and the GPU pipeline.
- **Focus:** [WebGL Artistry](https://frontendmasters.com/courses/creative-coding-webgl-shaders/).

### 3. The Book of Shaders
- **Status:** The fundamental bible of fragment shaders.
- **Why:** Learn to draw shapes and effects using only math. It is the cheapest (free) and most powerful skill for SATCORP’s "pulse" and "glow" effects.
- **Focus:** [The Book of Shaders](https://thebookofshaders.com/).

### 4. GSAP (GreenSock) - Advanced ScrollTrigger
- **Status:** Interaction Masterclass.
- **Why:** For scrollytelling. Understanding how to link 3D camera paths to the scrollbar.
- **Focus:** [Learning Center](https://gsap.com/learning/).

### 5. Shadertoy & GLSL Sandbox
- **Status:** Community Labs.
- **Why:** Reverse-engineer the best shaders in the world. Search for terms like "neural," "tactical," "grid," and "scanner" to find inspiration and code snippets.
- **Focus:** [Shadertoy](https://www.shadertoy.com/).

### 6. Gray Area Creative Code Intensive
- **Status:** Cohort-based professional transition.
- **Why:** For a holistic shift into creative technology, installations, and AI-integrated UI.
- **Focus:** [Gray Area SF](https://grayarea.org/).
