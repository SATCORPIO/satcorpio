# UPGRADE_PLAN // SATCORP_IO_2026
## Senior Architect's Vision & Recommendations

> [!CAUTION]
> A "massive upgrade" can be disruptive. I've designed this plan to be implemented in **Sprints**, minimizing downtime while maximizing the "WOW" factor of the SATCORP "Midnight Neural" ecosystem.

### PHASE 1: THE FOUNDATION (THE NEURAL GRID)
- **Goal:** Replace static background colors with a "GPU Heartbeat."
- **Action:** Implement a **Global Shader Background**. 
- **Code Tip:** Use `react-three-fiber` and `Three.js` to create a `NeuralGrid` component in `src/components`. A fragment shader should drive a subtle, glowing "Pulse" that follows the mouse orbitally.
- **Impact:** Immediate 100% increase in "Premium" perception.

### PHASE 2: TACTICAL TELEMETRY (DATA FLOW)
- **Goal:** Make the "Dossier" and project pages (Namtar, Frostheim) feel like active interfaces.
- **Action:** **Scrollytelling & Data Reveal.**
- **Code Tip:** Use `GSAP ScrollTrigger` to animate page sections. When a user scrolls to "OPERATIONS," items shouldn't just "fade in." They should "decrypt" — white-to-green text transitions, rapid flickering of data strings before stabilizing.
- **Impact:** Communicates "classified" and "high-tech" branding effectively.

### PHASE 3: SPATIAL NAVIGATION (COMMAND HUB)
- **Goal:** Redesign the main navigation or "Home" experience as a 3D interface.
- **Action:** **Immersive Project Carousel.**
- **Code Tip:** Replace the standard grid/carousel with a **3D Wheel** or **Starfield Map**. Each project node (Pulse, Kyrax, etc.) floats in space. The user clicks a node to "fly" into its dossier.
- **Impact:** Differentiates SATCORP from 99.9% of websites.

### PHASE 4: AUDIOMETRY (SOUND DESIGN)
- **Goal:** Full sensory immersion.
- **Action:** **Ambient Soundscape & Feedback UI.**
- **Code Tip:** Integrate a low-frequency ambient "hum" (8 bit/lo-fi/dark-synth). Use `Tone.js` for lightweight interaction bips and blips that are pitch-perfect for the "Neural" theme.
- **Impact:** Completes the "Military-Tech terminal" atmosphere.

### PHASE 5: NEURAL AI-UX (ADAPTIVE INTERFACE)
- **Goal:** The interface "learns" from the user.
- **Action:** **AI-Personalized "Dossier" HUD.**
- **Code Tip:** Track which sections (Frostheim, Pulse) the user spends time on. On the next visit, the "Primary HUD" dynamically prioritizes that data.
- **Impact:** Feels "alive" and truly "2026-standard."
